# Health Intelligence System v0.1.0

This is the first public prerelease of the SIP-aligned Health Intelligence System excellence pack.

## Included

- Cancer Intelligence module for screening prep, diagnostic navigation, treatment discussion prep, second-opinion packet assembly, and follow-up planning.
- Safety, privacy, validation, and clinical/legal review gate documents.
- Clinician handoff, evidence ledger, side-effect log, screening plan, diagnostic brief, treatment decision brief, and follow-up templates.
- Release manifest with checksums for package validation.

## Status

This release is a preclinical public prerelease. It is not medical advice, diagnosis, treatment recommendation, medication guidance, or a replacement for licensed clinicians.

Production use requires the review gate in `REVIEW-GATE.md` to be completed and logged.
