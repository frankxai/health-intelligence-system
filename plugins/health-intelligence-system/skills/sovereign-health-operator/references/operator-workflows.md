# Operator Workflows

## First Install Loop

1. Read the repo safety boundary.
2. Choose privacy mode.
3. Create private vault folders.
4. Copy private vault manifest, command center, record index, medication/supplement inventory, wearable ingestion manifest, weekly review, doctor visit prep, and clinician handoff templates.
5. Add prompt pack only after deciding what can leave the local vault.
6. Run privacy preflight before any cloud prompt.

## Weekly Health Operator Loop

Use private logs and wearable exports to produce:

- pattern summary;
- sleep/recovery notes;
- nutrition and hydration notes;
- training load and movement notes;
- energy and stress notes;
- questions for clinician, dietitian, trainer, or therapist;
- one low-risk wellness experiment.

No diagnosis, treatment, medication, supplement, or rehab advice.

## Doctor Loop

Use a 72-hour rhythm:

- gather records;
- update medication and supplement inventory;
- write timeline;
- create top three questions;
- prepare handoff export;
- update record index after the visit.
