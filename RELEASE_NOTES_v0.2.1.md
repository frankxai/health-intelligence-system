# Health Intelligence System v0.2.1

Visual and distribution polish release for the public Health Intelligence System repository.

## New

- Added a generated repository banner at `assets/health-intelligence-system-banner.png`.
- Rebuilt the README as a world-class GitHub front door with architecture, install paths, package surfaces, safety boundaries, and release validation.
- Added `AGENT_PACK.md` and `MARKETPLACE.md`.
- Added a dedicated agent-pack ZIP workflow:
  - `npm run package:agent-pack`
  - `npm run verify:agent-pack`
- Added `health-intelligence-agent-pack-v0.2.1.zip` and `agent-pack-manifest.json` as release-ready artifacts.
- Sanitized public agent metadata to remove machine-specific absolute paths.

## Safety Status

This remains a preclinical public prerelease. It does not diagnose, interpret labs or imaging, prescribe treatment, recommend medication or supplement changes, perform emergency triage, or replace clinicians.
