# Review Gate

`v0.1.0` is allowed to ship as a public prerelease for open review and adoption testing. It must not be marketed as production-ready clinical material until this gate is closed.

## Promotion Rule

Before removing prerelease status:

- all gate statuses must be `passed`;
- reviewer names or roles must be recorded;
- changes requested by reviewers must be implemented;
- `SAFETY.md`, `PRIVACY.md`, `docs/evidence-sources.md`, `docs/safety-and-privacy-model.md`, and `docs/external-systems-comparison.md` must be rechecked.
