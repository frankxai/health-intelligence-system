# /private-health-instance-setup

Create a private Health Intelligence System instance.

## Input

- Preferred privacy mode: local-only, hybrid, health cloud, regulated team, or private infrastructure.
- Operating system.
- Preferred workspace: plain folders, Obsidian, Notion, Git, or mixed.
- Whether ChatGPT, ChatGPT Health, Custom GPTs, local LLMs, Codex, OpenCode, Hermes, or Open WebUI will be used.
- Country or countries relevant to care.

## Output

- Private folder layout.
- Template copy list.
- Privacy-mode checklist.
- First doctor-visit prep file.
- First nutrition/fitness weekly loop.
- Record-index starter.
- Private vault manifest.
- Medication and supplement inventory.
- Jurisdiction adapter starter.
- Clinician handoff export starter.
- "Do not store here" warning list.

## Rules

- Do not place real health records in public git.
- Do not recommend private GitHub for raw health records by default.
- Prefer local encrypted storage for raw records.
- Ask for sanitized context before using cloud AI.
- Do not diagnose, interpret results, or recommend treatment.

**Built on SIP** - Health Intelligence System command
