# Contributing

Health contributions require more care than normal docs.

## Rules

- Do not submit real patient data.
- Do not add diagnosis or treatment recommendations.
- Prefer official health sources and guideline bodies.
- Add an evidence-check date for source-dependent content.
- Keep claims conservative and clinician-routed.
- Update [VALIDATION.md](VALIDATION.md) when adding a new workflow.

## Pull Request Checklist

- [ ] No private health data.
- [ ] No medical advice.
- [ ] Sources linked.
- [ ] Evidence-check date updated.
- [ ] Safety language preserved.
- [ ] Release package still builds.
